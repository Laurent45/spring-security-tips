# Spring Security Tips

A personal knowledge base project for learning and practicing Spring Security concepts. Each branch represents a specific concept or feature being explored.

## Running the Application

```bash
./mvnw spring-boot:run
```

Access at https://localhost:8080

**Test credentials:** `user` / `password`
